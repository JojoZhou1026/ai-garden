最近有几个朋友来问我：

“我平时用过豆包、Kimi、DeepSeek，也听说过‘小龙虾’，但其实不知道这些工具到底差在哪。我想让AI真实地帮助到我的日常工作和生活，到底该怎么开始？”

普通人开始用AI，第一关不是学习技术知识，也不是研究一堆模型参数，而是先迈出第一步：选择一个足够好的工具，把它安装好、注册好，并真正开始用起来。

当然，你可能会问：市面上那么多AI产品，到底该选哪个？

我的理念很简单：既然要花时间学，就优先学最好的那一档。下面这份准备清单，就以我目前最推荐的工具之一——Claude Code——为例。

你现在用豆包，大概率还是“问一句、答一句”。这没问题，适合写文案、总结资料、改邮件。但Claude Code不是一个更高级的聊天框，它更像一个能进你电脑工作区、看文件、改项目、陪你做东西的AI搭档。

所以，从豆包走到Claude Code，中间不是“换个软件”这么简单。

**但也不用把它想得太玄。先把下面几件事准备稳，你就已经走出第一步了。**

## 1. 电脑别太弱

如果准备换电脑，Mac M系列优先。不是因为它高贵，而是少折腾。性能、续航、环境兼容性都更省心。

Windows也能用，但建议16G内存起步。电脑太旧不是不能学AI，而是你会在安装、运行、开多个窗口时频繁卡住。新手最怕这种卡：你以为是自己笨，其实是设备在拖后腿。

## 2. 网络要稳定，不要天天换节点

Claude这类海外AI服务，对网络稳定性要求很高。你需要一个稳定工具，节点建议固定，优先新加坡。

重点不是“哪个节点最快”，而是“别乱跳”。今天新加坡，明天日本，后天美国，新账号很容易触发验证或风控。

如果用iPhone，有些工具可能需要海外Apple ID下载，提前准备。公司电脑和公司网络要谨慎，别把敏感资料随手丢给任何外部AI。

## 3. 先注册Claude，别急着上Claude Code

很多人听到Claude Code，就想马上装工具、跑命令、建项目。我的建议相反：先用网页版Claude。

先把它当普通聊天AI用几天。让它帮你改邮件、总结文章、拆任务、解释概念。先熟悉它怎么思考，再让它进入你的工作流。

你连Claude普通聊天都还没用顺，就直接上Claude Code，很容易变成“每一步都看不懂，但每一步都很高级”。

这不是学习，这是给自己制造挫败感。

## 4. 账号优先用Gmail

注册邮箱建议用Gmail。海外服务兼容性更好，验证码、登录提醒、订阅通知都更顺。

别用太冷门的邮箱，也别频繁换邮箱试注册。AI入门阶段，少折腾账号，多做真实任务。

## 5. 养号很重要

这里的“养号”不是玄学，本质是让账号像正常用户。

注册后别当天就买订阅，别马上高强度使用，别频繁切网络节点。先用免费版几天，每天打开聊两句，做点正常任务。

比如：

- 让它帮你改一封邮件
- 让它总结一篇文章
- 让它解释一个你不懂的概念
- 让它帮你拆一个工作计划

这一步很重要。很多人不是不会用AI，是账号还没稳定，就开始猛操作，最后被验证、限流、登录异常搞崩心态。

## 6. 订阅先Pro就够了

刚入门，先Pro。20美元/月左右，具体以官网显示为准。

**新手真正缺的不是额度，是使用习惯。**你先连续用两周，确认自己每天真的会打开，真的会拿它解决问题，再考虑要不要升级。

如果通过第三方渠道买订阅，注意账号安全。不要把主邮箱密码交出去，不要让别人长期代管账号，也不要在来路不明的账号里放公司资料。

便宜不是最大风险，失控才是。

## 最后：AI入门不是买工具，是换工作方式

很多人以为开始AI，就是买一个更强的模型。不是。

**真正的开始，是你每天拿一个真实问题交给AI一起做。**

不是问“你能干什么”，而是直接给它任务：

- 这封邮件帮我改得更有分寸
- 这个会议纪要帮我整理成行动项
- 这个方案帮我挑逻辑漏洞
- 这个概念我没听懂，用人话讲给我
- 我想做一个小工具，你陪我从0搭起来

当你开始这样用，AI才从“聊天玩具”变成“工作搭档”。

所以，对零基础朋友，我的建议很简单：

- 先把电脑、网络、账号、订阅准备稳
- 先用Claude网页版养几天手感
- 再进入Claude Code，不要反过来

Claude Code不是起点，它是你开始认真用AI做事之后，自然会走到的下一站。

## 写在最后

其实怎么开始不重要。

重要的是先开始。

后面的问题都会迎刃而解。
